# مصحف التدبر 📖

تلاوة تدبرية لجميع سور القرآن الكريم — واجهة عصرية مع خلفية Three.js تفاعلية

## التشغيل

```bash
npm install
npm run dev
```

ثم افتح المتصفح على: http://localhost:3000

## المكتبات المستخدمة

- **Next.js 14** — إطار العمل
- **React 18** — مكتبة الواجهة
- **Three.js** — الخلفية التفاعلية ثلاثية الأبعاد
- **Lucide React** — الأيقونات

## هيكل المشروع

```
quran-tadabbur/
├── app/
│   ├── layout.js        # تخطيط التطبيق الرئيسي
│   ├── page.js          # الصفحة الرئيسية
│   └── globals.css      # الأنماط العامة
├── components/
│   ├── IslamicBackground.jsx   # خلفية Three.js
│   ├── OpenQuranBook.jsx       # كتاب القرآن المفتوح
│   ├── OpenTadabburBook.jsx    # كتاب التدبر المفتوح
│   ├── HeartCard.jsx           # بطاقة رسالة من القلب
│   ├── QuranIndex.jsx          # فهرس السور
│   ├── QuranReader.jsx         # قارئ القرآن
│   ├── TadabburView.jsx        # عرض التدبر
│   └── HomePage.jsx            # الصفحة الرئيسية
└── data/
    ├── surahs.js        # بيانات السور ١١٤
    └── surahPages.js    # بيانات صفحات التدبر
```

## المزايا

- 🌿 خلفية Three.js تفاعلية تستجيب لحركة الماوس
- 📖 كتابان مفتوحان — القرآن والتدبر — بمحتوى حي متناوب
- 🔍 فهرس كامل لجميع ١١٤ سورة مع بحث
- 💌 تخصيص المحتوى باسم المستخدم
- 🎨 ألوان ذهبية وخضراء احترافية
- 📱 متجاوب مع جميع الأجهزة


🤝 كيف تساهم معنا؟ (Contribution Guide)
نحن نرحب بمساهماتكم سواء كانت برمجية أو إثراء للمحتوى التدبري:

1. إثراء بيانات التدبر (Data Entry)
بيانات التدبر محفوظة في المسار data/surahPages.js. لإضافة تدبر سورة جديدة، يرجى اتباع التنسيق التالي:

2. التطوير البرمجي (Development)
نطمح لتطوير الميزات التالية:

إضافة مشغل صوتي (Audio Player) للآيات.

تحسين أداء عناصر Three.js للأجهزة الضعيفة.

إضافة ميزة "الوضع الليلي" المتقدم.

تحويل المشروع إلى تطبيق ويب تقدمي (PWA).

📝 خطوات إرسال المساهمة
قم بعمل Fork للمشروع.

أنشئ فرعاً جديداً (git checkout -b feature/YourFeatureName).

قم بعمل Commit لتعديلاتك (git commit -m 'Add some feature').

ارفع الفرع (git push origin feature/YourFeatureName).

افتح Pull Request للمراجعة.

📂 هيكل المجلدات
app/: صفحات التطبيق الرئيسية والتنسيق العام.

components/: المكونات التفاعلية (Three.js، الكتب، الفهرس).

data/: قاعدة بيانات السور ونصوص التدبر.

public/: الأصول الثابتة (الصور، النماذج ثلاثية الأبعاد).
