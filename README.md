# مصحف التدبر 📖

تلاوة تدبرية لجميع سور القرآن الكريم — واجهة عصرية مع خلفية Three.js تفاعلية

## التشغيل

```bash
npm install
npm run dev
```

ثم افتح المتصفح على: http://localhost:3000

## المكتبات المستخدمة

- **Next.js 14** — إطار العمل
- **React 18** — مكتبة الواجهة
- **Three.js** — الخلفية التفاعلية ثلاثية الأبعاد
- **Lucide React** — الأيقونات


## المزايا

- 🌿 خلفية Three.js تفاعلية تستجيب لحركة الماوس
- 📖 كتابان مفتوحان — القرآن والتدبر — بمحتوى حي متناوب
- 🔍 فهرس كامل لجميع ١١٤ سورة مع بحث
- 💌 تخصيص المحتوى باسم المستخدم
- 🎨 ألوان ذهبية وخضراء احترافية
- 📱 متجاوب مع جميع الأجهزة

## 🚀 البدء السريع (بدون API Key)

يمكنك تطوير المشروع كاملاً **بدون** مفتاح Anthropic:

```bash
# 1. استنسخ المشروع
git clone https://github.com/your-repo/quran-tadabbur.git
cd quran-tadabbur

# 2. ثبّت التبعيات
npm install

# 3. أنشئ ملف البيئة بوضع Mock
echo "NEXT_PUBLIC_USE_MOCK=true" > .env.local

# 4. شغّل السيرفر
npm run dev
```

بهذا الإعداد ستعمل التطبيق بـ **Mock Data** جاهزة للسور: الفاتحة، الإخلاص، الناس.

## ✨ كيف تُضيف سورة جديدة؟

### الطريقة ١ — إضافة Mock Data (للتطوير)

افتح `data/mockTadabbur.js` وأضف:

```js
// رقم السورة كـ key
42: {
  title:        "عنوان السورة (3-5 كلمات)",
  topics:       ["موضوع 1", "موضوع 2", "موضوع 3"],
  thematicFlow: [
    "الفكرة الأولى مرتبطة بالحياة اليومية",
    "الفكرة الثانية",
    "الفكرة الثالثة",
  ],
  insights: [
    { label: "اسم قصير", desc: "شرح عاطفي عميق" },
    { label: "اسم آخر",  desc: "شرح آخر" },
  ],
  practical:    "تطبيق عملي واحد",
  heartMessage: "رسالة شخصية للقارئ",
},
```

### الطريقة ٢ — إضافة صفحات السورة (بيانات المصحف)

افتح `data/surahPages.js` وأضف في `SURAH_PAGES`:

```js
// رقم السورة
42: [
  {
    page:     497,   // رقم الصفحة في المصحف
    juz:      25,    // رقم الجزء
    verses:   "1-3", // نطاق الآيات
    title:    "اسم الوجه",
    text:     "نص الآيات كاملاً...",
    topics:   ["موضوع 1", "موضوع 2"],
  },
  // ... بقية الوجوه
],
```

## 🔒 معايير جودة المحتوى الشرعي

لضمان بقاء التطبيق في الإطار الصحيح:

- ✅ لا تضف تفسيرات خارج المأثور عن السلف
- ✅ الرسائل القلبية تكون في إطار التشجيع والتدبر — لا الفتوى
- ✅ استخدم مصادر موثوقة: تفسير ابن كثير، الطبري، السعدي
- ❌ لا تضف محتوى مُولَّداً بالكامل من AI بدون مراجعة بشرية
